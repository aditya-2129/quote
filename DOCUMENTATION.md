# Project Documentation Links

This file provides a centralized list of official documentation for the frameworks, libraries, and tools used in this project.

## Core Frameworks & Tools

- **[Tauri v2](https://v2.tauri.app/)** - Build cross-platform desktop apps with web technologies.
- **[React](https://react.dev/)** - The library for web and native user interfaces.
- **[Vite](https://vitejs.dev/)** - Next generation frontend tooling.
- **[TypeScript](https://www.typescriptlang.org/docs/)** - Strongly typed programming language that builds on JavaScript.
- **[Rust](https://doc.rust-lang.org/book/)** - A language empowering everyone to build reliable and efficient software.

## Frontend Libraries & Styling

- **[React Router v7](https://reactrouter.com/)** - Declarative routing for React.
- **[Tailwind CSS v4](https://tailwindcss.com/docs)** - A utility-first CSS framework.
- **[Three.js](https://threejs.org/docs/)** - JavaScript 3D library (used for CAD viewing).
- **[Lucide React](https://lucide.dev/docs/lucide-react)** - Beautiful & consistent icons.
- **[pdf-lib](https://pdf-lib.js.org/)** - Create and modify PDF documents (used for quote generation).
- **[occt-import-js](https://github.com/kovacsv/occt-import-js)** - Open CASCADE Technology (OCCT) STEP/IGES/BREP importer.
- **[clsx](https://github.com/lukeed/clsx)** & **[tailwind-merge](https://github.com/dcastil/tailwind-merge)** - Utilities for managing Tailwind classes.

## Backend (Rust) Dependencies

- **[Serde](https://serde.rs/)** - Serialization/Deserialization framework for Rust.
- **[Serde JSON](https://docs.rs/serde_json/latest/serde_json/)** - JSON support for Serde.
- **[Log](https://docs.rs/log/latest/log/)** - Logging facade for Rust.
- **[tauri-plugin-log](https://v2.tauri.app/plugin/log/)** - Tauri-specific logging integration.

## Development & Tooling

- **[ESLint](https://eslint.org/)** - Find and fix problems in your JavaScript code.
- **[Prettier](https://prettier.io/)** - An opinionated code formatter.
- **[npm](https://docs.npmjs.com/)** - Package manager for JavaScript.
- **[Cargo](https://doc.rust-lang.org/cargo/)** - Rust's package manager and build tool.
